# macOS 简洁计时器需求文档

## 项目概述
一个极简的 macOS 计时器应用，支持正计时和倒计时功能。

## 核心功能

### 1. 计时器模式
- **正计时（秒表）**：用户不输入时间，直接开始计时，从 0 开始累加
- **倒计时**：用户输入分钟数，开始倒计时
- 无需模式切换，根据用户是否输入数字自动判断

### 2. 交互方式
- 直接输入数字：用户在时间显示区域直接输入分钟数（如输入 25，点击开始后倒计时 25 分钟）
- 播放/停止按钮：使用符号图标，鼠标悬停时显示
- 关闭按钮：鼠标悬停时才显示，位于窗口左侧
- 暂停/继续功能：暂不实现（可选后续添加）

### 3. 提醒功能
- 倒计时结束时播放系统默认提示音
- 正计时无提醒

## 界面设计

### 窗口特性
- **无边框**：不使用 macOS 默认窗口栏
- **可调整大小**：用户可手动缩放
- **最小尺寸**：支持缩小到 80x60 像素
- **可拖动**：用户可拖动窗口到任意位置
- **极简设计**：只显示必要元素
- **视觉效果**：背景半透明，时间文字黑色带阴影，增强可读性
- **边界提示**：浅色边框提示可交互区域

### 窗口行为
- **Always on top**：始终显示在所有窗口最前面
- **非侵入式**：不干扰系统窗口管理逻辑（类似画中画）
- **位置自由**：可停留在屏幕任意位置，包括 dock 附近
- **不显示 Dock 图标**：应用不在 Dock 栏显示图标

### 布局元素
- 横向布局：左中右三栏
- 关闭按钮（最左侧，鼠标悬停时显示）
- 时间显示/输入区域（中间，直接输入数字）
- 播放/停止按钮（最右侧，使用符号图标，鼠标悬停时显示）

## 技术实现

### 技术栈
- **Swift + SwiftUI**
- 原生 macOS 应用
- 无额外依赖

### 关键技术点
- `NSWindow` 配置：无边框、always-on-top
- 窗口级别设置：`.floating` 或 `.popUpMenu`
- 最小窗口尺寸限制：80x60 像素
- 系统音效：`NSSound.beep()` 或系统提示音

## 非功能需求
- 应用体积小
- 启动快速
- 资源占用低
- 界面响应流畅

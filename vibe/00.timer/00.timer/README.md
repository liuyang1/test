# Timer - 极简 macOS 计时器

## 功能特性
- ⏱️ 正计时（秒表）和倒计时
- 🎯 极简界面，支持缩小到 80x60 像素
- 📌 始终置顶显示
- 🔔 倒计时结束声音提醒
- ⌨️ 全局快捷键 ⌃⌘M 启动/停止计时器

## 使用方法

### 编译
```bash
./build.sh
```

### 运行
```bash
open Timer.app
```

### 操作说明
1. **正计时**：不输入任何数字，直接点击"开始"
2. **倒计时**：在输入框输入分钟数，点击"开始"
3. **停止**：点击"停止"按钮重置计时器
4. **关闭**：点击左上角 ❌ 按钮退出程序
5. **移动窗口**：拖动窗口任意位置
6. **调整大小**：拖动窗口边缘

## 技术实现
- Swift + SwiftUI
- 无边框窗口
- Always-on-top (floating level)
- 最小尺寸：80x60 像素
- 系统提示音

